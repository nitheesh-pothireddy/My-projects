# Nitheesh Pothireddy — Projects

A monorepo of personal projects spanning data engineering, machine learning, and applied GenAI.

## 🚀 Featured (current)

### 🦫 [ReadRover](./readrover) — Multi-agent research assistant
Ask a research question, get a citation-backed brief synthesised from arXiv papers in ~60s. LangGraph orchestration with sub-agents for search planning, PDF reading, critique, and synthesis. Includes an LLM-as-judge eval harness over 5 reference topics.

**Stack:** LangGraph · RAG · ChromaDB · Sentence-BERT · Anthropic · Streamlit
**Status:** 🚧 active development

---

### 🔧 [ToolSmith](./toolsmith) — LoRA fine-tune for tool-calling
Teach a 0.5B Qwen model to call tools as well as a frontier model with a small LoRA fine-tune on the Open-Meteo API. Synthetic data generation, training on a free Colab T4, and a 4-axis evaluation comparing base vs. fine-tuned vs. Claude reference.

**Stack:** PyTorch · PEFT · TRL · 🤗 HuggingFace · Weights & Biases
**Status:** 🚧 active development
